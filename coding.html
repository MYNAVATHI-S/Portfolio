Scratch Projects & MIT App Inventor App

Assignment

Maze Game

The Maze Game challenges players to navigate through a maze while avoiding obstacles. Using arrow keys, the player moves a sprite to reach the goal. The game includes boundaries and logic to detect collisions.


Apple Catching Game

The Apple Catching Game is a fun challenge where a basket moves left and right to catch falling apples. Loops and conditional statements increase difficulty as the game progresses.


Fun Oreo Story

The Fun Oreo Story is an interactive storytelling project where characters engage in a playful Oreo adventure. It uses dialogues, sound effects, and animations to make the story engaging.

Attendance App Using MIT App Inventor

The MIT App Inventor Attendance App features a face camera to capture attendance along with the date and time. The app automatically stores the data and displays the captured image with the timestamp.

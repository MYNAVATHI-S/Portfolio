Food Share App UI/UX - Step by Step

Step 1: UI Design in Figma

Created the initial layout in Figma including home screen, navigation bar, food cards, and donate/request buttons. Focused on color harmony and mobile-first UI.

Step 2: Export Design Assets

Exported icons, buttons, and layout dimensions from Figma. Used consistent spacing and typography guidelines.

Step 3: HTML Structure

Created the layout using semantic HTML5 tags. Included header, cards, and a navigation bar for mobile responsiveness.

Step 4: CSS Styling

Styled the layout using Flexbox and media queries. Focused on colors matching Figma and modern neumorphic box shadows.

Step 5: JavaScript Interactions

Added dynamic effects like request button functionality, navigation highlight, and expandable steps like this page.

Step 6: Testing & Iteration

Tested responsiveness on different screen sizes and refined colors, spacing, and animations based on user feedback.
